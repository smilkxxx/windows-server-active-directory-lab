# Active Directory Domain Services (Windows Server 2012 R2)

## Descripción

Este laboratorio consiste en la implementación de un entorno empresarial utilizando Windows Server 2012 R2 como Controlador de Dominio Principal (Domain Controller).

Durante el laboratorio se configuró Active Directory Domain Services (AD DS), DNS y la estructura organizacional de una empresa ficticia mediante Unidades Organizativas (Organizational Units).

---

## Tecnologías

- Windows Server 2012 R2
- Active Directory Domain Services (AD DS)
- DNS
- PowerShell
- Windows 10
- VirtualBox

---

## Objetivos

- Instalar Active Directory.
- Configurar el primer Domain Controller.
- Configurar el servicio DNS.
- Crear Unidades Organizativas (OU).
- Administrar usuarios.
- Simular la estructura organizacional de una empresa.

## Actividades realizadas

- Instalación de Windows Server 2012 R2.
- Promoción del servidor a Domain Controller.
- Configuración del dominio.
- Instalación y configuración de DNS.
- Creación de cinco Organizational Units.
- Creación de veinticinco usuarios.
- Administración de usuarios y grupos.
- Validación del funcionamiento del dominio.
